### doc-video

The `<doc-video>` element is used to display a video with controls.

#### Attributes

- `src`: The source URL of the video.
- `type`: The MIME type of the video (defaults to `video/mp4`).

#### Example

```html
<doc-video src="https://youtu.be/vJ3jGgCrz2I"></doc-video>
```
