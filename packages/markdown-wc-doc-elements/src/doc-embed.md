## `<doc-embed>`

The `<doc-embed>` element is used to display an image with a caption. 

### Attributes

- `src`: The URL of the markdown-wc to embed.

### Example

```markdown
---
custom_elements:
  doc-embed: 'link/to/doc-embed.js'
---

<doc-embed 
  src="https://my-cool-markdown-wc-url.com/cool-markdown-wc.md">
</doc-embed>
```
